# MindLog v1.4.48

> AI 기반 감정 케어 다이어리

## 새로운 기능 ✨

- **빈 상태 안내 화면**: 일기를 처음 작성할 때 친절한 안내 화면이 표시됩니다
- **분석 단계 표시**: AI 분석 중 단계별 진행 상황을 실시간으로 확인할 수 있습니다

## 개선사항 🔧

- **글자수 카운터**: 일기 입력 중 실시간 글자수가 표시됩니다
- **색상 일관성 개선**: 앱 전체 색상이 테마 토큰으로 통일되어 다크/라이트 모드 모두 자연스럽습니다
- **애니메이션 토큰**: `AppDurations`를 도입해 애니메이션 지속 시간이 일관되게 관리됩니다

## 버그 수정 🐛

- **FCM 중복 알림 완전 해결**: 마음케어 알림이 간헐적으로 두 번 울리던 문제를 Firestore pre-lock 패턴으로 해결했습니다
- **다크모드 그라디언트 깨짐 수정**: 라이트 테마에서만 보이던 배경 그라디언트가 다크모드에서도 자연스럽게 표시됩니다
