# Cheer Me 추천 영역 접근성 체크리스트

적용 범위: 응원 메시지 입력 바텀시트의 카테고리 칩, 추천 문구 카드, 하단 안내 텍스트.

## 대비(Contrast) 체크

- [ ] 칩 선택 상태 텍스트/아이콘 (`#245D79`) vs 배경 (`#EAF6FC`) 대비가 4.5:1 이상이다.
- [ ] 칩 미선택 상태 텍스트/아이콘 (`#6F8596`) vs 배경 (`#F3F8FC`) 대비가 4.5:1 이상이다.
- [ ] 추천 문구 선택 상태 텍스트 (`#245D79`) vs 배경 (`#E6F4FB`) 대비가 4.5:1 이상이다.
- [ ] 추천 문구 미선택 상태 텍스트 (`#5F7485`) vs 배경 (`#F7FAFD`) 대비가 4.5:1 이상이다.
- [ ] 하단 안내 텍스트 (`#7F8C9A`) vs 바텀시트 배경 대비가 4.5:1 이상이다.

## 상호작용/상태 체크

- [ ] 선택 상태는 배경색 + 1dp 보더 + 약한 그림자(blur 6, alpha 0.10)로 미선택과 즉시 구분된다.
- [ ] 미선택 상태는 과도한 강조 없이 충분히 읽히며, 선택 상태와 혼동되지 않는다.
- [ ] 칩과 추천 문구 터치 타깃이 최소 48dp 이상이다.

## 뷰포트/텍스트 스케일 체크

- [ ] 320/360/412dp 폭에서 오버플로우 없이 렌더링된다.
- [ ] 시스템 텍스트 스케일 1.3x에서 칩/추천 문구/하단 안내 텍스트가 잘리지 않고 읽힌다.
