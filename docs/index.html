당신의 일기를 분석하여 감정 상태를 파악하고,
따뜻한 위로의 메시지와 맞춤형 행동 지침을 제공합니다.

주요 기능

📝

일기 작성 및 저장

로컬 데이터베이스(SQLite)에 안전하게 저장되며, 날짜별로 일기를 관리하고 조회할 수 있습니다.

🤖

AI 감정 분석

Groq API (Llama 3.3)를 활용하여 실시간에 가까운 감정 분석을 제공합니다. 캐릭터 선택에 따라 공감 톤이 달라집니다.

📊

감정 통계 대시보드

감정 추이 차트, 감정 정원(🌱→🌻), 키워드 태그 클라우드로 나의 감정 패턴을 시각화합니다.

💬

맞춤형 추천

AI가 분석한 감정 상태에 기반하여 공감 메시지와 맞춤형 행동 지침을 제안합니다.

🔒

프라이버시 보호

모든 일기 데이터는 로컬 기기에만 저장되어 개인 정보가 안전하게 보호됩니다.

📱

반응형 UI

320dp~600dp+ 다양한 화면 크기에 최적화된 반응형 디자인으로 모든 기기에서 편안하게 사용할 수 있습니다.

최근 개선

v1.4.25에서는 위젯 분해, 네비게이션 표준화, 테마 일관성, NLP 필터 고도화를 진행했습니다.

Architecture

🧩 위젯 분해 & 패턴 추출

DiaryListScreen(340줄 감소)을 TappableCard, WriteFab, ExpandableText 등 재사용 가능한 공통 위젯으로 분해했습니다. Entity 표시 로직은 Extension 메서드로 분리하여 Clean Architecture를 준수합니다.

Navigation

🧭 go_router 네비게이션 통일

레거시 Navigator.push 호출을 go_router의 context.push/go로 전면 마이그레이션했습니다. 선언적 라우팅으로 딥링크 대응과 뒤로가기 동작이 일관됩니다.

UX

🗑️ Soft Delete & Undo 패턴

일기 삭제 시 즉시 리스트에서 제거하되 5초 내 되돌리기가 가능한 Undo SnackBar 패턴을 적용했습니다. Timer 생명주기 관리와 낙관적 업데이트로 안정적인 UX를 제공합니다.

NLP

🇰🇷 한국어 텍스트 필터 고도화

AI 응답의 한국어 품질을 향상시키는 4단계 NLP 파이프라인(문법 교정→존칭 정규화→CJK 제거→후처리)을 구축했습니다. 테스트 136건 추가로 품질을 검증합니다.

Theme

🎨 다크모드 색상 일관성

하드코딩된 Colors.white/black을 colorScheme.surface/onSurface로 전면 교체하여 다크모드에서도 자연스러운 색상을 보장합니다.
