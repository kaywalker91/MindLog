MindLog

AI 기반 감정 케어 다이어리

당신의 일기를 분석하여 감정 상태를 파악하고,
따뜻한 위로의 메시지와 맞춤형 행동 지침을 제공합니다.

Google Play에서 다운로드 GitHub에서 보기

주요 기능

📝

일기 작성 및 저장

로컬 데이터베이스(SQLite)에 안전하게 저장되며, 날짜별로 일기를 관리하고 조회할 수 있습니다.

🤖

AI 감정 분석

Groq API (Llama 3.3)를 활용하여 실시간에 가까운 감정 분석을 제공합니다. 캐릭터 선택에 따라 공감 톤이 달라집니다.

📊

감정 통계 대시보드

감정 추이 차트, 감정 정원(🌱→🌻), 키워드 태그 클라우드로 나의 감정 패턴을 시각화합니다.

💬

맞춤형 추천

AI가 분석한 감정 상태에 기반하여 공감 메시지와 맞춤형 행동 지침을 제안합니다.

🔒

프라이버시 보호

모든 일기 데이터는 로컬 기기에만 저장되어 개인 정보가 안전하게 보호됩니다.

📱

반응형 UI

320dp~600dp+ 다양한 화면 크기에 최적화된 반응형 디자인으로 모든 기기에서 편안하게 사용할 수 있습니다.

최근 개선

v1.4.27에서는 저녁 알림 추가, KST 시간대 수정, CI/CD Fastlane 통합을 진행했습니다.

Notification

🌙 저녁 마음케어 알림

기존 아침 9시 알림에 더해 저녁 9시 알림이 추가되었습니다. "오늘 하루는 어떠셨나요?", "마음 정리할 시간이에요" 등 하루를 마무리하는 따뜻한 메시지를 받아보세요.

Timezone

🕐 KST 시간대 정확도 개선

Firebase Functions가 UTC에서 실행되는 점을 고려하여 명시적 KST 변환 로직을 추가했습니다. Intl.DateTimeFormat을 사용해 한국 시간 기준으로 정확하게 동작합니다.

CI/CD

🚀 Fastlane 배포 통합

GitHub Actions에서 Fastlane 기반 배포로 전환했습니다. Flutter 빌드부터 Play Store 업로드까지 Fastfile에서 통합 관리하며, 스크린샷 자동 업로드도 지원합니다.

Idempotency

🔒 시간대별 중복 방지

아침/저녁 알림이 각각 독립적으로 중복 발송을 방지합니다. "2024-01-15_morning", "2024-01-15_evening" 형식의 키로 시간대별 발송 이력을 관리합니다.

Screenshots

📸 스크린샷 v3 개편

앱 스크린샷이 v3로 새롭게 업데이트되었습니다. 시맨틱 파일명과 Fastlane metadata 구조로 Play Store 스크린샷 관리가 자동화되었습니다.
